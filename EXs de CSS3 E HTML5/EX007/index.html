para HTML a semântica é tudo aquilo que da siginificado ao texto desde um endereço até um nome de autor, diferente de CSS que lida com formas e estilos de cores para dar visual ao texto escrito
O HTML entre tanto em versões anteriores tinha diversas tags < > para mudar os estilos porém elas viraram obsoletas com o tempo, para verifica se uma tag é obsoleta ou não verificamos no site WWW.W3C.ORG
