Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em negrito não semântico usando a TAG <b>
Nessa frase tenho um termo em destaque semântico com a TAG <strong>

Itálico / Enfâse

Nessa frase temos um termo em itálico não semântico com a tag <i>
Nessa frase temos um termo em enfâse usuando a tag <em>

Texto marcado

Podemos também criar um texto marcado usando a tag <mark>

E no outro parágrafo temos outro texto marcado

Podemos notar que mesmo mudando a cor de um texto marcado, não significa que ele mudara de todos
Isso só é possível com um estilo em CSS com a finalidade disso ou mudar a cor de um por um

para isso criamos dentro de <head> um <style> para mudarmos a cor de qualquer <mark>

Texto Grande e Pequeno

estamos criando um texto grande tag <big> não semantica, e um pequeno neste parágrafo com a tag <small> semantica

Texto Deletado

podemos marcar um texto como excluído para indicar que ele deve ser lido mas nao considerado tag <del>

Texto Inserido

podemos marcar um texto como incluido para marcar uma enfâse que ele foi adicionado depois tag <ins>

Existem também texto sublinhados com a Tag <U> porémn essa técnica é não semântica!!!!

Texto Sobrescrito

Para inserir coisas do tipo x20+3 usar a tag <sup>

Texto Subscrito

Para inserir coisas do tipo H20 aqui usar a tag <sub>
